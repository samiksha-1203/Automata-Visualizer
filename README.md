# Automata Visualizer

An interactive web-based tool to create, visualize, and simulate **DFA and NFA**.

## 🚀 Features
- Create automata visually
- Step-by-step string simulation
- NFA → DFA conversion
- DFA minimization

## 🛠 Tech
HTML, CSS, JavaScript (Canvas)

## ▶️ Usage
Open `index.html` in your browser.
